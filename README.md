# 速度打字挑戰 (Speed Typing Challenge)

一個使用 Python 與 Tkinter 製作的打字練習遊戲。

## 功能
- 使用 Tkinter 建立圖形化介面 (GUI)
- 多種難度選擇
- 倒數計時
- 分數計算
- 正確率 (Accuracy) 計算
- 每分鐘字數 (WPM) 計算
- 即時字元顏色回饋
- 遊戲結束後在同一個視窗顯示結果

## 執行方式
```bash
git clone https://github.com/Lilyyeh126/typing-game.git
cd typing-game
uv sync
uv run main.py
